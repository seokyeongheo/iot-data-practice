\begin{itemize}
\tightlist
\item
  활성화 함수란?

  \begin{itemize}
  \tightlist
  \item
    활성화 함수(activation function)은 입력 신호의 총합을 출력 신호로
    변환하는 함수입니다.
  \item
    활성화 함수는 입력 신호 총합의 활성화 여부를 결정하는 역할을 합니다.
  \end{itemize}
\end{itemize}

\[a = b + w_1x_1 + w_2x_2\] \[y = h(a)\]

\begin{itemize}
\item
  활성화 함수의 종류

  \begin{enumerate}
  \def\labelenumi{\arabic{enumi}.}
  \tightlist
  \item
    시그모이드 함수(sigmoid function)

    \begin{itemize}
    \tightlist
    \item
      시그모이드 함수는 비선형 함수로 아래와 같은 식을 지니는
      함수입니다. 시그모이드 함수는 특정 임계값을 기준으로 출력을
      분류합니다.
    \item
      입력값이 크거나 작을 때 - 시그모이드 함수에서 1 혹은 0으로
      수렴하기 시작하는 부분 - 는 그래디언트값이 지나치게 작아지고,
      연산이 무거워집니다.
    \item
      범위가 {[}0, 1{]} 이기 때문에 그래디언트가 양수 혹은 음수인지에
      따라 loss에 대한 가중치의 그래디언트가 양수 혹은 음수가 됩니다.
      \[h(x) = \frac{1}{1 + exp(-x)}\]
    \end{itemize}
  \item
    하이퍼탄젠트 함수(Hypertangent function)

    \begin{itemize}
    \tightlist
    \item
      하이퍼탄젠트 함수는 그레디언트 감소 현상(gradient vanishing
      problem)을 줄이는 활성화 함수입니다.
    \item
      시그모이드 함수는 범위가 0부터 1까지인데 반하여, 하이퍼탄젠트
      함수는 -1부터 1까지 입니다.
      \[tanh(a) = \frac{e^a - e^-a}{e^a + e^-a} = 2\sigma(2a) - 1\]
      \[tanh' (x) = 1 - tanh^2 (x)\]
    \end{itemize}
  \item
    렐루 함수(Rectified Linear Unit - ReLU function)

    \begin{itemize}
    \tightlist
    \item
      렐루 함수는 입력이 0을 넘으면 그 입력을 그대로 출력하고, 0 이하면
      0을 출력하는 함수로 아래와 같은 식을 지닙니다.
      \[h(x) = \begin{cases} x \; (x > 0) \\ 0 \; (x <= 0) \end{cases}\]
    \item
      가중치의 총합 a가 큰 경우에도 기울기가 1로 유지되기 때문에, a가
      커도 그레디언트 감소 현상이 발생하지 않습니다.
    \item
      미분도 편리하여 계산복잡도가 낮습니다.\\
    \item
      CNN처럼 레이어의 수가 많은 경우에 유용합니다.
    \end{itemize}
  \item
    Leaky 렐루

    \begin{itemize}
    \tightlist
    \item
      Leaky 렐루는 렐루 함수의 그래디언트가 죽는 현상을 방지하기 위해
      제안되었으며 아래의 식과 같습니다. \[f(x) = \text{max}(0.01x, x)\]
    \item
      x가 음수일 때 그래디언트가 0.01배라는 것을 제외하고는 렐루 함수와
      같습니다.
    \end{itemize}
  \end{enumerate}
\end{itemize}
